Java Basics ☕ (🚧 In Arbeit)

Ich möchte einen Einblick in die Programmiersprache Java geben. Was gleich zu beginn auffällt ist das es eine Objektorientierte Programmiersprache ist, aber das werden wir noch sehen.

Java Abkürzungsjungel 🌴

Um ein Java Programm zu starten zu bekommen braucht es einen Laufzeitumgebung denn Java läuft in einer virtuellen Maschine der sogenannten JVM (Java Virtual Maschine) die Laufzeitumgebung trägt den Namen JRE (Java Runtime Enviroment). Und zusätzlich weil wir hier ja auch Programmen entwickeln und nicht nur ausführen brauchen wir noch das JDK (Java Development Kit)

Endlich Kompilieren und Ausführen 🚀

Als Kompilieren wir der Prozess des übersetzten von der Menschen leserlichen Programmiersprache zur Maschinensprache bezeichnet bzw. in Java zum sogenannten Bytecode.

Der Compiler heißt javac und wird wie folgt aufgerufen: javac HalloWelt.java Dieser erstellt dann den Bytecode für uns er ist mit dem Kürzel .class gekennzeichnet, die ersten Bytes sind Hex dargestellt die Worte cafe babe. ☕

Zum ausführen verwendet man einfach den Befehl java aber aufpassen wir wollen zwar die .class ausführen müssen aber genau die Endung weg lassen also: java HalloWelt

Seit Java 11 gibt es auch die Möglichkeit sein Quellcode direkt mit dem Befehl java auszuführen, also so: java HalloWelt.java

Typen und Variablen

Java ist eine statisch typisierte Sprache das heißt jede Variable hat einen definierten Type. Dieser ist schon zur Compiletime bekannt, deswegen statisch. Eine Variabeln deklaration hat folgenden Aufbau: Datetyp Variablenname = Wert;

In Java gibt es zwei Arten von Datentypen, die primitiven und Referenzdatentypen.

Primitive Datentypen

Zu den primitiven Daten Typen gehören die Zahlen, Zeichen und Wahrheitswerte.

Ganzzahlen

Es gibt verschieden Größen mit entsprechenden Wertebereichen:

Datentyp Bit Minimum Maximum
Byte 8-Bit -128 127
short 16-Bit -32.768 32.767
int 32-Bit -2.147.483.648 2.147.483.647
long 64-Bit -9.223.372.036.854.775.808 9.223.372.036.854.775.807
Fließkommazahlen
Datentyp Bit Minimum Maximum
float 32-Bit -3.4 * (10^38) 3.4 * (10^38)
double 64-Bit -1.7 * (10^308) 1.7 * (10^308)

Gleitkommazahlen sind bei den hinteren Stellen nicht mehr genau.

Hier ein kleines Beispiel mit int und double die wahrscheinlich am häufigsten verwendet werden.

  • Typen.java 
public class Typen {
       public static void main (String[] args) {
             // Ganzzahlen
             int number0;
             number0 = 15;               // 15
             number0 += 10 + 5 * 3;       // 15 + 10 + (5 * 3) = 40
             System.out.println(number0);

            // Gleitkommazahlen
             double number1;
             number1 = 6.7876;

       }
}

Umwandeln von Zahlen

Widening (Erweitern) ist das Umwandeln einer Variable von einem Typ mit kleinem Wertebereich in eine Variable von einem Typ mit größeren Wertebereich.

Narrowing (Eingrenzen) ist das Umwandeln einer Variable von einem Typ mit großen Wertebereich in eine Variable von einem Typ mit kleinen Wertebereich. Wobei hier aufgepasst werden muss dass der ursprüngliche Wert der Variable im neuen Wertebereich auch dargestellt werden kann.

Casting ist die explizite Angabe das der Typ der Variable umwandeln möchten.

Das siehr dann so aus: 

long foo = 5;
int bar = (int) foo;

Character

Sind einzelne Zeichen und sehen in Java so aus: 

char c = 'C';
Diese werden im Unicode encodiere.

Warheitswerte

Haben nur zwei Werte: true oder false. Diese werden häufig auch als Boolesche Werte bezeichnet.

Beispiel: 

boolean t = true;
boolean f = false;

Wrapperklassen

Sind Objektversionen der Primitiven Datentypen, diese heißen: Byte, Short, Integer, Long, Float, Double, Boolean und Character.

Beispiel am Integer:

Integer zahl = Integer.valueOf(5);
Integer zahl = Integer.valueOf("8");

Es ist gängige Praxis die Methode valueOf aufzurufen. Dies kann auch String Strings in denn Datentype parsen.

Zwei weitere bekannte Methoden sind MIN_VALUE und MAX_VALUE, diese geben jeweils den kleinsten und grüßen Wert des jeweiligen Datentypes aus.
